# Contributing Policy

This repository is public for visibility, bug reporting, and feedback only.

## What Is Welcome

- Bug reports with clear reproduction steps
- Gameplay feedback and balancing suggestions
- Writing feedback about presentation, pacing, or polish
- Technical observations that help identify defects or regressions

## What Is Not Accepted

- Pull requests that change code, maps, dialogue, graphics, audio, or other project files
- Reuploads, redistributions, or altered versions of the project without written permission
- Automated mass-edits, mirror repositories, or derivative packs based on this repo

## How To Share Feedback

Open a GitHub issue with:

1. A clear title
2. The chapter, map, or system involved
3. Steps to reproduce, if reporting a bug
4. Screenshots or video when useful
5. The result you expected and the result you observed

Issues may be used for discussion and planning, but repository ownership and implementation decisions remain with the maintainer.
