  Frame 2: Standing straight

SKELETON/BONE STRUCTURE (Simple)
================================================================================

For realistic articulation in Aseprite:

HEAD (Parent)
  ├─ Eye_L (child)
  └─ Eye_R (child)

TORSO (Parent)
  ├─ Arm_L (child)
  │   └─ Hand_L (child of Arm_L)
  ├─ Arm_R (child)
  │   └─ Hand_R (child of Arm_R)
  ├─ Leg_L (child)
  │   └─ Foot_L (child of Leg_L)
  └─ Leg_R (child)
      └─ Foot_R (child of Leg_R)

This allows you to:
- Rotate head independently
- Move arms naturally
- Bend legs for jumping
- Create fluid animations

TIPS FOR DRAWING
================================================================================

1. START SIMPLE
   - Outline the shape first (black lines)
   - Fill with base color
   - Add details/shading last

2. CONSISTENCY
   - Keep proportions same across all frames
   - Same line thickness for outlines
   - Consistent eye style

3. GRID ALIGNMENT
   - Use Aseprite grid (View > Grid)
   - Set grid to 8x8 or 16x16
   - Helps with clean pixels

4. ONION SKIN
   - View > Onion Skin to see previous/next frames
   - Helps with smooth animation flow
   - Check for jumping/jerking

5. FRAME TIMING
   - Idle/normal: 100-150ms
   - Quick movements: 50-75ms
   - Slow effects: 150-200ms

6. EXPORT FORMAT
   - File > Export As: [name]_{frame}.png
   - Creates: sans_npc_0.png, sans_npc_1.png, etc.
   - Ready for sprite atlas integration
